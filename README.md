timesheet-nodejs-mongo
======================

INIT (populateDB)

http://localhost:3000/init

URL : 


get('/ts', GET ALL);
get('/ts/:id', GET BY OBJECTID);
get('/ts/:user/:day', GET BY USER AND DAY);
post('/ts', CREATE NEW timesheet);
put('/ts/:id', UPDATE timesheet);
delete('/ts/:id', DELETE timesheet);
